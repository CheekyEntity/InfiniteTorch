# Infinite Torch - Cave Crawlers Mod

A simple quality-of-life mod for Cave Crawlers that grants torches infinite duration. No more worrying about your light source burning out in the deep dark.

## Features
* **Infinite Energy**: All torches (held and wall-mounted) are infused with infinite energy.
* **Frozen Duration**: Stops the torch duration loss multiplier entirely.
* **Welcome Notification**: Displays a one-time "Torch Blessed" popup in-game so you know the mod loaded correctly.

## Installation
1. Ensure you have **[BepInEx 6.0.0 (IL2CPP)](https://builds.bepinex.dev/projects/bepinex_be)** installed for Cave Crawlers.
2. Download the latest `InfiniteTorch.dll`.
3. Place the `.dll` file into your game's directory at:
   `...\SteamLibrary\steamapps\common\Cave Crawlers\BepInEx\plugins\`
4. Launch the game and enjoy the light!
